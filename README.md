polyx0x_001
===========
WebGL music video for "Polyop - run_grids".

[See it running](http://polyop.uk/run_grids)

[Check out the EP on Bandcamp](https://polyop.bandcamp.com/album/run-test-polyx0x-001)


## Quick start
```
$ yarn run dev
```

And view the project at `localhost:8080`
